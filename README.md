# VR2AR Converter 2

> [!WARNING]
> This repository is obsolete and was replaced by https://github.com/michael-mueller-git/vr2ar-converter-v3

Convert your adult VR Videos into Passthrough AR Videos.

Difference to my [vr2ar-converter-v1](https://github.com/michael-mueller-git/vr2ar-converter) is other platform (docker) and the method for the mask generation.

## Usage

Use the provided container and deploy on device with nvida gpu. Then use the grad.io webui to convert your videos.
